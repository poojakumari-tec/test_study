### 1. マルチLLMプロバイダー対応poojakumari

- OpenAI（GPT-4 Turbo）

- Deepseek（Deepseek Chat）

### 2. Webインターフェース (`app.py`)

- Streamlitベースの使いやすいUI

### 3. 必要なパッケージのインストール

```bash
pip install -r requirements.txt
pip install google-generativeai
pip install openai
```
